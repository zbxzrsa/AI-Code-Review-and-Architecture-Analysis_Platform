# 技术更新日志模板

## 记录要点
- 版本标签：v1/v2/v3 与具体发行号
- 变更摘要：技术/组件/模型更新点
- 验证结果：三重 AI 验证结论、压测与用户场景模拟结果
- 性能对比：基线 vs 本次（延迟/吞吐/资源/错误率）
- 发布窗口：预生产与正式放量时间、放量策略（蓝绿/金丝雀）
- 回滚预案：镜像/配置快照、数据库迁移方式、触发条件
- 审计：签名（v1/v3/v2）、责任人、日期

## 示例条目
- 版本：v2.4.0
- 变更：引入新模型路由策略 + 编码模型升级
- 验证：v1 三周期通过；三重 AI 均签名；压测 p99 -18%，CPU -12%
- 发布：蓝绿，30%→60%→100% 逐级；监控阈值见运行手册
- 回滚：保留 v2.3.1 镜像与配置，DB 兼容迁移（双写）
- 审计：ai.sign.v1/v3/v2 均 PASS，日期 2025-xx-xx
